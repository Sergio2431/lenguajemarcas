Los tres pilares de la Web

Tim Berners-Lee es considerado el padre de la Web porque desarrolló los tres básicos para el funcionamiento de la Web

  • HTML
  • HTTP
  • URL

HTML

HTML, siglas de HyperText Markup Language, hace referencia al lenguaje de marcado para la elabroración de páginas web. Es un estándar que sirve de referencia para la elabroración de páginas web en sus diferentes versiones, define una estructura básica y un código (denominado código HTML) para la definición de contenido de una página web, como texto, imágenes, entre otros. Es un estándard a cargo de la W3C, organización dedeicada a la estandarización de casi todas las tecnologías ligadas a la web, sobre todo en lo referente a su escritura e interpretación. Es el lenguaje con el que se define las páginas web.

Fuente: HTML, Wikipedia

HTTP

HyperText Transfer Protocol o HTTP (en español protocolo de transferencia de hipertexto) es el protocolo usado en cada transacción de la Word Wide Web. HTTP fue desarrollado por el Word Wide Web Consortium y la Internet Engineering Task Force, colaboración que culminó en 1999 con la publicación de una serie de RFC, el más importante de ellos es el rfc 2616 que especifica la versión 1.1.

Fuente: HTTP, Wikipedia

HTTP

Un localizador de recursos uniforme o URL -siglas en ingles de Uniform Resource Locator- es un identificador de recursos uniforme (URI) cuyos recursos referidos pueden cambiar, esto es, la dirección puede apuntar a recursos variables en el tiempo. Están formados por una secuencia de caracteres, de acuerdo a un formato modélico y estándar, que designa recursos en una red, como Internet.
